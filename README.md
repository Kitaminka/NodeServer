# NodeServer
Node.js web server using only default [**http module**](https://nodejs.org/api/http.html).
___
### Installation
- Clone the repository:
```bash
git clone https://github.com/kitaminka/node-server.git
```
- Start the program:
```bash
npm start
```
___
### Features
- Automatic routes
- Simple template system
- Error 404 page
